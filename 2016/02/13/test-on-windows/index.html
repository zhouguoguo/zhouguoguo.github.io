2016年2月13日

2016年2月7日，羊年的最后一天，猴宝宝专属年就要到了~

材料准备

今天做的是红枣酸奶戚风蛋糕。
材料：
鸡蛋 4个
原味酸奶 100g
低筋面粉 80g
细砂糖 50g
食用油 30g
玉米淀粉 5g
红枣碎 20g左右
柠檬汁 几滴

动手开做

  1. 鸡蛋的蛋黄和蛋白分开在两个大盆里，蛋黄盆里加入食用油，酸奶，拌匀

  2. 筛入低筋面粉，用刮刀搅拌至无颗粒

  3. 蛋白里滴入几滴柠檬汁，加入玉米淀粉

  4. 打发蛋白，分三次加入细砂糖，打发蛋白至硬性发泡

  5. 取一半蛋白霜加入蛋黄糊中，翻拌均匀，混合之后，再倒回剩下的蛋白霜里，翻拌均匀

  6. 翻拌均匀后加入红枣碎，翻拌均匀

  7. 准备八寸活底蛋糕模具，将蛋糕糊倒入

  8. 用刮刀把表面抹平

  9. 入烤箱中层125°，60-70分钟,完成

成品

时间烤长收腰了T_T
